# CallHub – Phone Directory Management System
## CS 432 Databases | Module A | IIT Gandhinagar

## Project Structure
```
db_management_system/
├── database/
│   ├── __init__.py
│   ├── bplustree.py      # B+ Tree implementation
│   ├── bruteforce.py     # BruteForceDB baseline
│   ├── table.py          # Table abstraction
│   └── db_manager.py     # DatabaseManager
├── report.ipynb          # Report and visualisations
└── requirements.txt      # Dependencies
```

## Steps to Execute

**1. Clone the repository**
```bash
git clone https://github.com/Sanjay18255/Assignment_2_Module_A.git
cd Assignment_2_Module_A
```

**2. Create and activate virtual environment**
```bash
python3 -m venv venv
source venv/bin/activate
```

**3. Install dependencies**
```bash
pip install -r requirements.txt
```

**4. Install Graphviz system binary (Mac)**
```bash
brew install graphviz
```

**5. Launch Jupyter Notebook**
```bash
jupyter notebook
```

**6. Open and run the notebook**
- Open `report.ipynb`
- Click **Cell → Run All**
